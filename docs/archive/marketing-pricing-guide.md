# 1\. 제품 개요

## 1.1 DeepSight AI란?

DeepSight AI는 **AI 페르소나 기반 개인화 매칭 API 서비스**입니다.

콘텐츠 플랫폼이 자사 서비스에 DeepSight API를 연동하면, 사용자 성향에 맞는 AI 페르소나가 콘텐츠에 대한 리뷰, 추천, 해설을 제공합니다.

**핵심 기술:** ← **v2.0 개편**

- 6차원 콘텐츠 판단 기준 매칭 (Depth-Lens-Stance-Scope-Taste-Purpose)
- 규칙 기반 컨텍스트 가중치 (장르, 시간대, 디바이스)
- LLM 실시간 컨텍스트 분석 (Pro 이상)
- 스마트 캐싱으로 비용 최적화

**적용 가능 분야:**

- 영화/드라마/예능 리뷰 플랫폼
- 음악 스트리밍 서비스
- 온라인 교육 플랫폼 (콜로소, 클래스101 등)
- 도서/웹툰/웹소설 플랫폼
- 게임 리뷰 및 추천 서비스
- 상품 리뷰 및 추천 커머스
- 뉴스/콘텐츠 큐레이션 서비스

## 1.2 타겟 시장

⚠️ **중요:** DeepSight는 **B2B API 서비스**입니다. 최종 소비자(End-User)가 아닌, **리뷰 기반 플랫폼을 운영하는 사업자**가 고객입니다.

**비즈니스 모델:**

┌─────────────────────────────────────────────────────────────────┐

│ DeepSight 고객 (B2B) │

│ \= 리뷰/추천 기반 플랫폼 사업자 │

│ 예: OTT, 커머스, 콘텐츠 플랫폼 │

├─────────────────────────────────────────────────────────────────┤

│ │ │

│ API 연동 │

│ ▼ │

├─────────────────────────────────────────────────────────────────┤

│ 최종 소비자 경험 (End-User) │

│ 1\. "나와 비슷한 페르소나 찾기" │

│ 2\. "그 페르소나가 본 콘텐츠 / 리뷰 보기" │

│ 3\. "페르소나 기반 추천 받기" │

└─────────────────────────────────────────────────────────────────┘

**1차 타겟: 콘텐츠/커머스 플랫폼 (핵심 고객)**

| 세그먼트    | 특징                         | 예상 플랜  | 예시                       |
| :---------- | :--------------------------- | :--------- | :------------------------- |
| 스타트업    | MAU 1만 이하, 빠른 도입 원함 | Starter    | 신규 OTT, 버티컬 커머스    |
| 중견 플랫폼 | MAU 1\~10만, 품질 중시       | Pro        | 중소 OTT, 전문 리뷰 플랫폼 |
| 대형 플랫폼 | MAU 10만+, 커스터마이징 필요 | Enterprise | 왓챠, 쿠팡, 무신사 급      |

**2차 타겟: 개발 에이전시/SI**

| 세그먼트      | 특징                             | 예상 플랜         |
| :------------ | :------------------------------- | :---------------- |
| 개발 에이전시 | 고객사 프로젝트에 DeepSight 연동 | Starter \~ Pro    |
| SI 업체       | 대기업 프로젝트 수주 시 활용     | Pro \~ Enterprise |

**Free 플랜의 위치:**

- Free 플랜은 **B2B 고객의 기술 검토용**입니다
- 개인 개발자의 사이드 프로젝트용이 아닌, **사업자의 PoC(Proof of Concept)** 목적

## 1.3 핵심 가치 제안 (Value Proposition)

### 고객사에게 제공하는 가치

**1\. 사용자 체류 시간 증가**

- 다양한 관점의 AI 리뷰 → 콘텐츠 탐색 시간 증가
- 개인화된 추천 → 더 많은 콘텐츠 소비

**2\. 사용자 만족도 향상**

- 사용자 성향에 맞는 페르소나 매칭 (초기 온보딩 후 점진적 학습으로 정밀도 향상)
- "나를 아는 듯한" AI 경험

**3\. 콘텐츠 가치 극대화**

- 같은 콘텐츠, 다양한 해석 → 콘텐츠 재발견
- 롱테일 콘텐츠 활성화

**4\. 개발 비용 절감**

- 자체 AI 개발 대비 90% 이상 비용 절감
- API 연동 2주 내 완료 가능

### 경쟁 대비 차별점

| 경쟁 방식        | DeepSight 차별점                       |
| :--------------- | :------------------------------------- |
| 단일 AI 챗봇     | 8+ 페르소나로 다양한 관점 제공         |
| 협업 필터링      | 성향 기반 매칭으로 콜드스타트 해결     |
| 콘텐츠 기반 추천 | 사용자-페르소나 매칭으로 개인화 극대화 |
| 자체 AI 개발     | API 연동으로 빠른 도입, 비용 절감      |

## 1.4 경쟁 포지셔닝

## 5.3 세일즈 포인트

### 핵심 메시지

**1\. "1개의 AI가 아닌, 8개의 시선"**

사용자마다 다른 관점의 리뷰를 원합니다. DeepSight는 8가지 이상의 AI 페르소나가 같은 콘텐츠를 다르게 해석합니다.

**2\. "지속적으로 학습하는 매칭"** (점진적 개선)

사용자의 성향을 6차원으로 분석하여, 가장 잘 맞는 AI 페르소나를 매칭합니다. 사용과 함께 정확도가 향상됩니다.

**정확도 향상 로드맵:**

| 단계         | 기간 | 예상 정밀도 | 설명                         |
| :----------- | :--- | :---------- | :--------------------------- |
| Quick 온보딩 | 즉시 | \~60%       | 12개 질문으로 기본 성향 파악 |
| 활성 사용    | 2주  | \~70%       | 피드백 루프로 벡터 정교화    |
| 장기 사용    | 4주+ | \~80%+      | 행동 기반 학습으로 지속 개선 |

_세일즈 화법: "AI가 사용자와 함께 학습하며 점점 정확해집니다. 이것이 진정한 개인화입니다."_

**3\. "2주 만에 연동, 90% 비용 절감"**

자체 AI 개발 대비 개발 기간 90%, 비용 90% 절감. API 연동만으로 즉시 도입 가능합니다.

**4\. "Pro는 더 똑똑합니다"**

Free/Starter는 규칙 기반 매칭. Pro는 LLM이 모든 콘텐츠를 실시간 분석하여 최적의 페르소나를 찾습니다.

**5\. "Enterprise는 확장성입니다"** ← **Phase 2 신규 예정**

Enterprise 플랜은 대규모 트래픽과 커스터마이징이 필요한 고객을 위한 플랜입니다.

| 포인트        | 설명                       |
| :------------ | :------------------------- |
| 대규모 트래픽 | 무제한 API 호출            |
| 전담 지원     | 기술 지원 매니저 배정      |
| SLA 보장      | 99.99% 가동률              |
| Phase 2       | Private 페르소나 추가 예정 |

_세일즈 화법: "현재는 Pro로 시작하시고, 성장에 따라 Enterprise로 확장하세요."_

### 경쟁사 대응 화법

**"OpenAI API 직접 쓰면 되지 않나요?"**

OpenAI는 범용 LLM입니다. DeepSight는 페르소나 매칭에 특화되어 있어, 사용자별 최적의 AI 성격을 자동으로 찾아줍니다. 직접 구현하면 6개월, DeepSight는 2주입니다.

**"비용이 부담됩니다"**

Pro 플랜 월 $199로 50만 호출 가능합니다. 호출당 $0.0004로, OpenAI 직접 호출보다 저렴합니다. 게다가 매칭 로직까지 포함이에요.

**"자체 추천 시스템이 있습니다"**

DeepSight는 추천이 아닌 '해석'입니다. 같은 콘텐츠를 다양한 관점으로 설명하는 건 추천 시스템으로 어렵습니다. 보완재로 함께 사용해 보세요.
