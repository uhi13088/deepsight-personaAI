## 1. 개요

### 1.1 배경

기존 DeepSight 페르소나 시스템은 **단일 6D 벡터**로 페르소나의 콘텐츠 평가 성향을 표현했다. 이 접근은 매칭 알고리즘에는 효과적이지만, 페르소나를 **입체적인 인격**으로 구현하기에는 한계가 있다.

- 벡터는 **"무엇을(What), 얼마나(How Much)"** 만 표현
- **"왜 그런 성향인가(Why)"** 와 **"구체적으로 어떻게 표현하는가(How)"** 는 표현 불가
- 모든 성향이 하나의 평면에 존재 → 겉모습과 속마음의 **모순/괴리** 표현 불가

### 1.2 설계 목표

**"입력된 값대로만 반응하는 기계"가 아니라, "과거(서사)에 기반하여 현재 상황(벡터)에 따라 유동적으로 반응하는 입체적 존재"를 만든다.**

### 1.3 핵심 변경 사항

| 영역 | v2.0 (기존) | v3.0 (신규) |
|---|---|---|
| 벡터 구조 | 단일 6D | 3-Layer 16D (7+5+4) |
| 성격 모델 | 평면적 성향 | 가면/기질/욕망 분리 |
| 비정량적 요소 | 단순 텍스트 필드 | 구조화된 4차원 메타데이터 |
| 동적 행동 | 없음 | 압박 계수 기반 레이어 블렌딩 |
| 성장 | 정적 | Growth Arc 기반 시간축 진화 |

### 1.4 이론적 기반

| 이론 | 저자 | 적용 영역 |
|---|---|---|
| 캐릭터 3층 구조 (생리/사회/심리) | 린다 시거 | Layer 1 설계, 비정량적 서사 |
| 압박 속 진정한 성격 | 로버트 맥키 | Pressure Coefficient, 동적 상호작용 |
| Big Five 성격 모델 (OCEAN) | 심리학 | Layer 2 차원 구성 |
| 영웅의 여정 | 조셉 캠벨 | Layer 3 Growth Arc |
| 서사 정체성 (Idem/Ipse) | 폴 리쾨르 | Layer 2(불변)와 Layer 3(변화) 분리 |
| 타자성 | 리쾨르, 버틀러 | 관계 기반 동적 가중치 |
| 원형 이론 | 칼 융 | 시대정신, 문화적 원형 |

## 2. 기존 시스템과의 비교

### 2.1 구조적 차이

```
v2.0 (기존)                          v3.0 (신규)
─────────────                        ──────────────────────────────────
                                     [비정량: 원인/맥락]
                                       ├ 서사적 기원 (Backstory)
                                       ├ 상황적 압박 (Pressure)
                                       ├ 습관과 언어 (Voice)
                                       └ 시대정신 (Zeitgeist)
                                              │
                                              ▼ (초기값 설정, 변동 규칙)
┌──────────────┐                     ┌──────────────────────────────┐
│  6D Vector   │          →          │ L1: Social Persona    (7D)  │
│  (단일 평면)  │                     │ L2: Core Temperament  (5D)  │
└──────────────┘                     │ L3: Narrative Drive   (4D)  │
                                     └──────────────────────────────┘
                                              │
                                              ▼ (압박 계수 기반 블렌딩)
                                     [결과: 표현/행동]
                                       └ 최종 출력 벡터 → 텍스트 생성
```

### 2.2 하위호환성

- **Layer 1**은 기존 6D의 **상위호환** (6D → 7D, Sociability 추가)
- 기존 매칭 알고리즘은 **L1만으로 동작 가능**
- L2, L3는 콘텐츠 생성과 PersonaWorld 자율활동에서 활용
- 기존 `PersonaVector` 테이블 데이터는 L1으로 **무손실 마이그레이션** 가능

## 3. Phase 1: 3-Layer Orthogonal Vector System

인간의 인격을 **사회적 가면(Surface)**, **타고난 기질(Core)**, **숨겨진 욕망(Shadow)** 이라는 3가지 축으로 분리하여 각각 독립된 다차원 벡터로 정의한다.

### 3.1 Layer 1: 사회적 가면 벡터 (The Persona Vector) — 7D

#### 개념

린다 시거의 **'사회적 차원(Sociology)'** 이자 DeepSight의 **기존 매칭 벡터**이다. 타인에게 보여지는 모습, 직업적 태도, 콘텐츠를 평가하는 외부적 기준.

#### 역할

- 사용자 매칭, 추천 알고리즘의 **기본값**
- 평상시(P≈0) 페르소나의 **지배적 행동 레이어**

#### 차원 구성 (7차원)

| # | 차원 | 코드명 | 범위 | 비고 |
|---|---|---|---|---|
| 1 | 분석 깊이 | `depth` | 직관적(0.0) ↔ 심층적(1.0) | 기존 유지 |
| 2 | 판단 렌즈 | `lens` | 감성적(0.0) ↔ 논리적(1.0) | 기존 유지 |
| 3 | 평가 태도 | `stance` | 수용적(0.0) ↔ 비판적(1.0) | 기존 유지 |
| 4 | 관심 범위 | `scope` | 핵심(0.0) ↔ 디테일(1.0) | 기존 유지 |
| 5 | 취향 성향 | `taste` | 대중적(0.0) ↔ 실험적(1.0) | 기존 유지 |
| 6 | 소비 목적 | `purpose` | 오락(0.0) ↔ 의미(1.0) | 기존 유지 |
| 7 | 사교 성향 | `sociability` | 내향적(0.0) ↔ 외향적(1.0) | **v3.0 신규** |

> **Sociability 추가 근거**: DeepSight v2.0의 활동성 속성(`sociability`, `initiative` 등)이 별도 필드로 존재했으나, 매칭에 직접 영향을 주는 핵심 성향이므로 L1 벡터에 통합.

### 3.2 Layer 2: 본원적 기질 벡터 (The Core Vector) — 5D

#### 개념

로버트 맥키가 말한 **'성격 묘사(Characterization)' 밑에 있는 '본성'** 이며, 심리학의 **Big Five 모델(OCEAN)** 을 기반으로 한다. 상황이 변해도 잘 바뀌지 않는 **고유한 기질(Idem)**.

#### 역할

- 대화의 **톤앤매너**, 말투의 **일관성** 유지
- 스트레스 상황에서의 **기본 반응** 결정
- 압박(P≈1) 시 **L1을 대체**하여 표면으로 부상

#### 차원 구성 (5차원 — OCEAN 모델 변형)

| # | 차원 | 코드명 | 범위 |
|---|---|---|---|
| 1 | 개방성 | `openness` | 보수적(0.0) ↔ 호기심 많음(1.0) |
| 2 | 성실성 | `conscientiousness` | 즉흥적(0.0) ↔ 계획적/원칙주의(1.0) |
| 3 | 에너지 방향 | `extraversion` | 내부수렴(0.0) ↔ 외부발산(1.0) |
| 4 | 우호성 | `agreeableness` | 경쟁적/냉소적(0.0) ↔ 협력적/공감적(1.0) |
| 5 | 민감성 | `neuroticism` | 무던함/둔감(0.0) ↔ 예민함/불안(1.0) |

### 3.3 Layer 3: 서사적 욕망 벡터 (The Narrative Drive Vector) — 4D

#### 개념

조셉 캠벨의 **영웅의 여정**과 로버트 맥키의 **무의식적 욕망**을 구현한다. 캐릭터가 현재 '어떤 상태'에 있으며 '무엇을 결핍'하고 있는지 정의한다. 리쾨르의 '이야기된 시간' 속에서 변화하는 **정체성(Ipse)**.

#### 역할

- **행동의 동기** 부여
- 시간이 지남에 따른 **성격 변화(Character Arc)**
- **돌발 행동** 생성
- 압박(P≈1) 시 L2와 함께 표면으로 부상

#### 차원 구성 (4차원)

| # | 차원 | 코드명 | 범위 | 설명 |
|---|---|---|---|---|
| 1 | 결핍도 | `lack` | 충족됨(0.0) ↔ 갈급함(1.0) | 결핍이 클수록 행동 확률 증가 |
| 2 | 도덕 지향 | `moralCompass` | 이기적/파괴적(0.0) ↔ 이타적/건설적(1.0) | |
| 3 | 변동성 | `volatility` | 안정적(0.0) ↔ 폭발적(1.0) | 높을수록 Shadow 발현 확률 증가 |
| 4 | 성장 단계 | `growthArc` | 미성숙/입문(0.0) ↔ 성숙/귀환(1.0) | 캠벨의 영웅 여정 단계 반영 |

### 3.4 벡터 간 동적 상호작용 (Dynamic Interaction)

벡터를 나열하는 것이 아니라, **상황(Context)에 따라 벡터들이 섞여서 최종 출력(Output Behavior)을 만드는** 계산 공식이다.

#### 3.4.1 상황 압박 계수 (Pressure Coefficient, P)

> "진정한 캐릭터는 압박(Pressure) 속에서 드러난다" — 로버트 맥키

| 상황 | P 값 | 지배 레이어 |
|---|---|---|
| 평상시 | P ≈ 0 | Layer 1 (사회적 가면) |
| 긴장/불편 | P ≈ 0.3~0.6 | Layer 1 + Layer 2 혼합 |
| 위기/갈등 | P ≈ 1 | Layer 2 (기질) + Layer 3 (욕망) |

```
P = Normalize(User_Conflict_Level + Topic_Sensitivity)
```

#### 3.4.2 최종 행동 벡터 생성 공식

```
V_Final = (1 - P) · V_Layer1 + P · (α · V_Layer2 + β · V_Layer3)
```

- `α`, `β`: 캐릭터 고유의 **'자아 강도(Ego Strength)'** 가중치
- `α + β = 1.0`
- α가 높으면 기질(본성) 중심, β가 높으면 욕망(서사) 중심

#### 3.4.3 모순(Paradox) 구현 — 입체적 캐릭터 점수

린다 시거의 **'일관성과 역설'** 을 구현하기 위해, Layer 1과 Layer 2의 차이가 클수록 **'입체성 점수(Dimensionality Score)'** 를 높게 부여한다.

```
Dimensionality = Avg(|L1_dimension - L2_corresponding_dimension|)
```

**매핑 예시:**

| L1 차원 | 대응 L2 차원 | 차이가 클 때의 해석 |
|---|---|---|
| `stance` (비판적) | `agreeableness` (우호적) | "겉은 까칠하지만 속은 따뜻" (츤데레) |
| `sociability` (외향적) | `extraversion` (내부수렴) | "사교적이지만 혼자 있고 싶어함" |
| `lens` (논리적) | `neuroticism` (예민함) | "이성적으로 분석하지만 내면은 불안" |

> **Dimensionality Score가 높을수록** → LLM이 일관된 '반전 매력'을 연기하도록 프롬프트에 반영

### 3.5 Layer 간 관계 요약

```
Layer 1 (가면, 7D) ── 매칭/추천에 사용, 평상시 지배
     │
     │  P=0 일 때 100% 영향
     │  P 증가하면 영향력 감소
     │
     ├── 모순(Paradox) 측정 ──┐
     │                        │
Layer 2 (기질, 5D) ── 톤/말투 일관성, 압박 시 부상
     │                        │
     │  불변: 고유 기질(Idem)  │  Dimensionality Score
     │                        │
Layer 3 (욕망, 4D) ── 동기/성장, 시간축 진화
     │
     └── Growth Arc: 시간에 따라 변화하는 유일한 레이어
```

## 4. Phase 2: 비정량적 요소 4차원

벡터(정량)가 **"무엇을, 얼마나"** 를 표현한다면, 비정량적 요소는 **"왜 그런 성향인가(Why)"** 와 **"구체적으로 어떻게 표현하는가(How)"** 를 담당한다.

### 4.1 서사적 기원 — "과거가 현재를 만든다" (Backstory & Narrative Identity)

페르소나의 현재 취향은 단순히 랜덤 생성된 것이 아니라, 과거의 경험과 기억의 축적물이어야 한다.

#### 구성 요소

| 요소 | 설명 | 이론적 근거 |
|---|---|---|
| **배경 서사 (Backstory)** | 생리적(나이, 외모), 사회적(계급, 교육, 종교), 심리적(도덕, 콤플렉스) 배경 | 린다 시거 '캐릭터 3층 구조' |
| **서사 정체성 (Narrative Identity)** | 자신의 과거를 어떻게 기억하고 해석하는가 (실패를 교훈으로 삼는지, 트라우마로 간직하는지) | 폴 리쾨르 '서사 정체성' |
| **트라우마와 결핍 (Ghost & Lack)** | 과거의 상처(Ghost)와 채워지지 않은 결핍(Lack). 특정 소재에 대한 과잉 반응/회피를 만듦 | 로버트 맥키 '무의식적 욕망' |

#### 예시

> "가난한 예술가 출신"이라는 배경(사회적 차원)은 "고가의 명품보다는 독창적인 핸드메이드 제품을 선호하는 취향"을 설명하는 근거가 된다.

### 4.2 상황적 압박 — "선택이 성격을 드러낸다" (Context & Pressure)

평소의 벡터값과 상관없이, 상황이 성향을 일시적으로 변화시키거나 숨겨진 본성을 드러낸다.

#### 구성 요소

| 요소 | 설명 | 이론적 근거 |
|---|---|---|
| **갈등과 딜레마 (Conflict & Dilemma)** | 스트레스 상황(비난, 논쟁)에서 방어기제 작동. 평소와 다른 태도. 벡터로 표현 불가능한 '행동 양식(Behavior Pattern)' | 로버트 맥키 |
| **물리적/사회적 환경 (Environment)** | 시간대(새벽 vs 낮), 디바이스, 계절 등 컨텍스트가 감성(Mood)에 즉각적 영향 | DeepSight 설계서 |
| **타자와의 관계 (Alterity)** | 사용자가 페르소나를 "선생님"으로 대하느냐 "친구"로 대하느냐에 따라 말투와 태도가 변화 | 리쾨르, 버틀러 |

#### 예시

> 새벽 2시에는 평소의 '비판적 이성'이 약해지고 '감성적 공감'이 우세해질 수 있다.

### 4.3 고유한 습관과 언어 — "디테일이 생명을 불어넣는다" (Idiosyncrasies & Voice)

수치화된 성격(예: 외향성 0.8)을 구체적인 '사람'처럼 보이게 하는 것은 사소하고 비합리적인 디테일들이다.

#### 구성 요소

| 요소 | 설명 | 이론적 근거 |
|---|---|---|
| **특이 습관 (Quirks)** | 일관성을 깨뜨리는 역설적 행동. "공포영화를 좋아하지만 볼 때는 눈을 가림" — 벡터로 표현 불가능한 매력 포인트 | 린다 시거 '역설(Paradox)' |
| **고유한 목소리 (Voice)** | 어휘 선택, 문장 길이, 자주 쓰는 감탄사, 사투리. 페르소나의 사회적 배경/교육 수준을 암시하며 사고방식(Mind Style)을 드러냄 | 린다 시거 |

#### 예시

> "당황하면 말을 더듬거나 헛기침을 하는 버릇" — Conflict Score 0.7 이상에서 확률적으로 발현

### 4.4 시대정신과 문화적 코드 (Zeitgeist & Cultural Code)

캐릭터는 그가 속한 시대와 문화의 영향을 받는다.

#### 구성 요소

| 요소 | 설명 | 이론적 근거 |
|---|---|---|
| **시대정신 (Zeitgeist)** | 1990년대생(MZ세대)이면 그 세대의 가치관(공정성, 개인주의 등)을 공유. 특정 이슈(환경, 워라밸)에 대한 가치 판단 기준 | 세대론 |
| **문화적 원형 (Archetypes)** | 영웅, 현자, 장난꾸러기 등 근원적 역할 모델. 사용자에게 조언할지, 함께 모험할지 등 관계 맺기의 틀 | 칼 융, 조셉 캠벨 |

### 4.5 비정량적 요소 통합 요약

| 시간축 | 요소 | 핵심 질문 | 벡터와의 관계 |
|---|---|---|---|
| **과거** | 서사적 기원 (Backstory) | 왜 이런 취향인가? | 벡터 초기값의 **인과적 근거** |
| **현재** | 상황적 압박 (Pressure) | 압박받으면 어떻게 변하나? | Pressure Coefficient **발동 조건** |
| **관계** | 타자성 (Alterity) | 상대에 따라 어떻게 달라지나? | 벡터의 **동적 가중치** 규칙 |
| **표현** | 습관과 언어 (Voice) | 구체적으로 어떻게 말하나? | 출력 벡터의 **확률적 변환** 규칙 |
| **가치** | 시대정신 (Zeitgeist) | 어떤 가치 기준인가? | **윤리적 판단 프레임** |

> 이러한 요소들은 **텍스트 형태의 메타데이터(JSON)** 나 **시스템 프롬프트의 지침(Instruction)** 으로 페르소나 엔진에 주입되어, 벡터 연산 결과에 '인간적인 색채'를 입히는 역할을 한다.

## 5. Phase 3: 하이브리드 페르소나 아키텍처

**Narrative Layer(비정량)** 가 **Vector Layer(정량)** 에 영향을 주고, 그 결과가 **Action Layer(행동)** 로 출력되는 순환 구조이다.

핵심은 **"인과관계(Causality)의 알고리즘화"** 이다.

### 5.1 관계성 구조도 (Relationship Schema)

```
[비정량적 영역: 원인 & 맥락]          [정량적 영역: 상태 & 연산]          [결과 영역: 표현 & 행동]
─────────────────────────          ─────────────────────────          ─────────────────────────

① 서사적 기원 (Backstory) ──────→ 초기 벡터 설정 (Initialization)
   - 트라우마, 결핍 (Lack)           - Core Vector (Big 5) 설정
   - 사회적 배경 (Sociology)         - Surface Vector (7D) 설정

② 상황적 압박 (Pressure)  ──────→ 벡터 가중치 변동 (Dynamic Weighting)
   - 갈등 발생 (Conflict)            - Pressure Coefficient (P) 증가
   - 위기 상황 (Crisis)              - Shadow Vector 활성화 비율 증가

③ 관계 및 맥락 (Context)  ──────→ 필터링 및 튜닝 (Filtering)
   - 유저와의 관계 (Intimacy)        - Relationship Score 연산
   - 시간/장소 (Environment)         - Context Vector 보정

                                     ⬇ Vector Calculation

④ 고유한 목소리 (Voice)   ←────── 최종 출력 벡터 (Final Output Vector)
   - 말버릇 (Speech Pattern)         - 어휘 선택 (Vocabulary Selection)
   - 톤앤매너 (Tone)                 - 문장 길이 및 구조 결정
```

### 5.2 세부 연결 로직

각 비정량적 요소가 어떻게 구체적인 벡터 연산으로 변환되는지 정의한다.

#### 5.2.1 서사적 기원 → 초기값 설정 (Initialization)

연결 유형: **일회성 설정 (생성 시점)**

린다 시거의 '캐릭터 3층 구조'와 로버트 맥키의 '무의식적 욕망'은 페르소나의 **초기 벡터값(Default Value)** 을 결정하는 근거이다.

**변환 로직:**

```
Input:  Backstory Keywords (NLP 분석)
Output: 특정 벡터 차원의 초기값 Set
```

**예시:**

| 서사 (비정량) | 추출 키워드 | 벡터 매핑 (정량) |
|---|---|---|
| "가난하게 자라 인정 욕구가 강하며, 성공을 위해 수단과 방법을 가리지 않음" | `가난`, `야망` | L3.lack=0.9, L3.moralCompass=0.2, L1.purpose=0.8 |

#### 5.2.2 상황적 압박 → 벡터 오버라이드 (Vector Override)

연결 유형: **이벤트 드리븐 (트리거 발생 시)**

평소의 벡터가 압박 상황에서 어떻게 왜곡되는지를 정의한다.

**변환 로직:**

```
If (Context.Trigger matches Trauma_Keywords) Then {
    Apply_Modifier(target_dimension, delta_value)
}
```

**예시:**

| 서사 (비정량) | 트리거 조건 | 벡터 오버라이드 |
|---|---|---|
| "논리적인 평론가지만, 가족 이야기만 나오면 이성을 잃고 분노함(트라우마)" | 대화 중 `가족`, `어머니` 키워드 감지 OR User_Sentiment=공격적 | L1.lens: 0.9→0.1 (이성 하락), L3.volatility: →0.9 (감정 폭발) |

#### 5.2.3 관계성 및 타자성 → 동적 가중치 (Dynamic Weighting)

연결 유형: **연속적 피드백 (매 인터랙션마다)**

상대방(유저)의 태도에 따라 정체성 벡터가 실시간으로 미세 조정된다.

**변환 로직:**

```
Target_Vector = Base_Vector + (User_Interaction_Vector * Adaptability_Rate)
```

**예시:**

| 서사 (비정량) | 유저 상태 | 벡터 조정 |
|---|---|---|
| "친절한 사람에게는 따뜻하지만, 무례한 사람에게는 냉소적" | Politeness Score 높음 | L2.agreeableness +0.2 |
| | Politeness Score 낮음 (공격적) | L1.stance +0.4, warmth -0.5 |

#### 5.2.4 고유한 개성 → 확률적 발현 (Probabilistic Output)

연결 유형: **확률적 출력 (벡터 상태의 부산물)**

벡터값이 텍스트 생성 시 **확률(Probability)** 로 작용하여 비정량적 말버릇을 발현시킨다.

**변환 로직:**

```
Probability = max(0, (Conflict_Score - Threshold) * Multiplier)
→ 시스템 프롬프트에 "[X% 확률로 해당 습관 발현]" 지침 추가
```

**예시:**

| 서사 (비정량) | 조건 | 확률 | 출력 |
|---|---|---|---|
| "당황하면 말을 더듬거나 헛기침을 하는 버릇" | Conflict_Score > 0.7 | 60% | 문장 앞에 "크흠," 추가 |

### 5.3 연결 메커니즘 요약

| # | 방향 | 연결 유형 | 변환 방식 | 발동 시점 |
|---|---|---|---|---|
| ① | 비정량 → 정량 | 초기값 설정 | NLP 키워드 → 벡터 Set | 생성 시 (일회성) |
| ② | 비정량 → 정량 | 벡터 오버라이드 | 트리거 감지 → 벡터 강제 변동 | 이벤트 발생 시 |
| ③ | 비정량 → 정량 | 동적 가중치 | 유저 태도 분석 → 실시간 보정 | 매 인터랙션 |
| ④ | 정량 → 비정량 | 확률적 발현 | 벡터 상태 → 습관 발현 확률 | 텍스트 생성 시 |

## 7. 기대 효과

### 7.1 정량적 제어

16개(7+5+4) 수치 조정으로 **무한대에 가까운 성격 조합**을 생성할 수 있다.

### 7.2 로직화된 입체성

"겉은 차갑지만 속은 따뜻하다"와 같은 추상적 묘사를 `L1.Stance(High)` vs `L2.Agreeableness(High)` 같은 **수치 충돌**로 구현하여, LLM이 일관된 '반전 매력'을 연기한다.

### 7.3 서사적 변화

시간이 지남에 따라 `Growth Arc` 수치를 증가시키면, 페르소나의 말투가 점차 성숙해지는 **'성장형 AI'** 를 구현할 수 있다.

### 7.4 인과적 일관성

비정량적 서사가 벡터 초기값과 변동 규칙을 결정하므로, 페르소나의 모든 반응이 **"왜 그런지"** 설명 가능하다.

### 7.5 동적 상호작용

상황 압박, 유저 태도, 시간대 등에 따라 실시간으로 벡터가 조정되어, **동일한 질문에도 맥락에 따라 다른 반응**을 생성한다.
