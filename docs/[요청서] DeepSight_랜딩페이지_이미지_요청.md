> 작성일: 2026-02-06

## 1. 개요

DeepSight B2B 랜딩 페이지에 필요한 이미지/일러스트 목록입니다.
현재 플레이스홀더로 아이콘과 CSS로 대체되어 있으며, 추후 제작된 이미지로 교체가 필요합니다.

## 2. 필수 이미지 (Hero / 핵심 섹션)

### 2.1 6D 벡터 레이더 차트 일러스트

| 항목       | 내용                                                                                                              |
| ---------- | ----------------------------------------------------------------------------------------------------------------- |
| **위치**   | 6D 벡터란 무엇인가요? 섹션                                                                                        |
| **용도**   | 6개 차원을 시각적으로 보여주는 레이더 차트                                                                        |
| **크기**   | 480x480px (Retina 대응 960x960px)                                                                                 |
| **스타일** | - 깔끔한 선형 그래프 스타일<br>- 파란색-보라색 그라데이션<br>- 6개 축: Depth, Lens, Stance, Scope, Taste, Purpose |
| **참고**   | 사용자 A와 페르소나 B의 벡터를 겹쳐서 유사도를 시각적으로 표현                                                    |

**예시 레퍼런스:**

```
         Depth
           ▲
          /|\
         / | \
   Lens ●  |  ● Purpose
        \  |  /
         \ | /
          \|/
    Stance ● ─ ● Scope
             |
           Taste
```

## 11. 프롬프트 사용 시 주의사항

### 11.1 일관성 체크리스트

- [ ] 모든 이미지에 흰색 또는 매우 밝은 배경 사용
- [ ] 그라데이션은 Blue(#2563EB) → Indigo(#4F46E5) 기본
- [ ] 아바타는 모두 동일한 일러스트 스타일 유지
- [ ] 아이콘은 동일한 선 두께 (2-3px) 유지
- [ ] 그림자는 부드럽고 미묘하게 (blur 8-16px, opacity 10-15%)
- [ ] 모서리는 항상 둥글게 (8-16px radius)

### 11.2 품질 설정

| 항목              | 권장값                       |
| ----------------- | ---------------------------- |
| 해상도            | Retina (2x) 생성 후 리사이즈 |
| 스타일            | --style minimal              |
| 버전              | --v 6 이상                   |
| 품질              | --q 2 (고품질)               |
| 가로세로비 (--ar) | 위 프롬프트 참고             |

### 11.3 후처리

1. 생성된 이미지를 지정 크기로 리사이즈
2. PNG로 저장 (투명 배경 필요시)
3. SVG 변환 필요시 벡터화 도구 사용 (아이콘용)
4. 컬러 일관성 검토 후 필요시 조정

## 12. 임시 대체 방안

이미지가 준비되기 전까지는 다음으로 대체:

1. **Lucide 아이콘**: 오픈소스 아이콘 라이브러리 활용
2. **CSS 그라데이션**: 배경 효과
3. **플레이스홀더 박스**: 점선 테두리 + 설명 텍스트

현재 랜딩 페이지에는 위 방식으로 대체되어 있습니다.
