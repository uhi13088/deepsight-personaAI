# DeepSight 랜딩 페이지 이미지 요청서

> 작성일: 2026-02-06
> 상태: 이미지 제작 요청

## 1. 개요

DeepSight B2B 랜딩 페이지에 필요한 이미지/일러스트 목록입니다.
현재 플레이스홀더로 아이콘과 CSS로 대체되어 있으며, 추후 제작된 이미지로 교체가 필요합니다.

---

## 2. 필수 이미지 (Hero / 핵심 섹션)

### 2.1 6D 벡터 레이더 차트 일러스트

| 항목       | 내용                                                                                                              |
| ---------- | ----------------------------------------------------------------------------------------------------------------- |
| **위치**   | 6D 벡터란 무엇인가요? 섹션                                                                                        |
| **용도**   | 6개 차원을 시각적으로 보여주는 레이더 차트                                                                        |
| **크기**   | 480x480px (Retina 대응 960x960px)                                                                                 |
| **스타일** | - 깔끔한 선형 그래프 스타일<br>- 파란색-보라색 그라데이션<br>- 6개 축: Depth, Lens, Stance, Scope, Taste, Purpose |
| **참고**   | 사용자 A와 페르소나 B의 벡터를 겹쳐서 유사도를 시각적으로 표현                                                    |

**예시 레퍼런스:**

```
         Depth
           ▲
          /|\
         / | \
   Lens ●  |  ● Purpose
        \  |  /
         \ | /
          \|/
    Stance ● ─ ● Scope
             |
           Taste
```

| 항목       | 내용                                                                     |

| **내용**   | 1. 콜드스타트 질문 → 2. 6D 벡터 계산 → 3. 페르소나 매칭 → 4. 콘텐츠 추천 |

### 2.3 기존 추천 vs DeepSight 비교 인포그래픽

| 항목       | 내용                                                                                 |
| ---------- | ------------------------------------------------------------------------------------ |
| **위치**   | 기존 추천 시스템과 무엇이 다른가요? 섹션                                             |
| **용도**   | 두 시스템의 차이를 한눈에 보여주는 비교 그래픽                                       |
| **크기**   | 800x500px                                                                            |
| **스타일** | - 좌우 분할 레이아웃<br>- 왼쪽: 회색톤 (기존 시스템)<br>- 오른쪽: 파란색 (DeepSight) |
| **내용**   | - 협업 필터링 vs 6D 벡터 매칭<br>- "이것도 봤습니다" vs "이 페르소나가 추천합니다"   |

## 10. 임시 대체 방안

이미지가 준비되기 전까지는 다음으로 대체:

1. **Lucide 아이콘**: 오픈소스 아이콘 라이브러리 활용
2. **CSS 그라데이션**: 배경 효과
3. **플레이스홀더 박스**: 점선 테두리 + 설명 텍스트

현재 랜딩 페이지에는 위 방식으로 대체되어 있습니다.
